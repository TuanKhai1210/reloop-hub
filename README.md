# ReLoop Hub Backend

Backend API for the ReLoop Hub project.

## Technology stack

- FastAPI
- PostgreSQL
- SQLAlchemy
- Alembic
- Pydantic
- Psycopg

## FastAPI dashboard implementation

The complete dashboard API implementation is available in
[`fastapi-backend/`](fastapi-backend/README.md). It includes Smart RVM
telemetry and WebSocket updates, quality inspection, users and rewards,
collection-route optimization, material traceability, ESG reports, Docker
Compose, seed data, and end-to-end tests.
